# Web Scraping Indeed Job Listings with Selenium and BeautifulSoup

## Overview

This Python script utilizes Selenium and BeautifulSoup to scrape job listings from Indeed. It navigates through multiple pages of search results, extracts relevant job details, and cleans the data for analysis.

## Requirements

- Python 3.x
- Selenium
- BeautifulSoup
- Pandas
- Chrome WebDriver (or WebDriver suitable for your browser)
